개인정보처리방침

정부청사 구내식당 앱

1. 개요

정부청사 구내식당 앱은 사용자의 개인정보 보호를 최우선으로 합니다. 본 개인정보처리방침은 앱의 데이터 처리 방식을 명확하게 설명합니다.

주요 사항:

  • 본 앱은 정부청사 구내식당의 주간 식단표를 조회하는 앱입니다
  • 앱 자체적으로 개인정보를 수집하지 않습니다
  • 즐겨찾기 설정은 기기 내에만 저장됩니다
  • Google Mobile Ads를 통해 광고가 표시되며, 광고 ID 및 기기 정보가 수집될 수 있습니다

광고 서비스의 데이터 처리 방식을 이해하시려면 본 방침을 주의 깊게 읽어주시기 바랍니다.

2. 개인정보 수집

앱에서 수집하는 정보:

정부청사 구내식당 앱은 개인정보를 수집하지 않습니다. 다음 정보는 수집하지 않습니다:

  • 개인 식별 정보 (이름, 이메일, 전화번호)
  • 사용 데이터 또는 분석 정보
  • 위치 정보 또는 GPS 데이터
  • 연락처 또는 캘린더 데이터
  • 쿠키 또는 추적 기술

광고 서비스에서 수집하는 정보:

본 앱은 Google Mobile Ads를 사용하며, 광고 목적으로 다음 데이터가 수집될 수 있습니다:

  • 광고 ID (AD_ID) - 광고용 고유 기기 식별자
  • 광고 상호작용 데이터 (노출, 클릭, 전환)
  • 기기 정보 (모델, OS 버전, 화면 크기)
  • 대략적인 위치 파악을 위한 IP 주소

이 데이터는 본 앱이 아닌 Google과 광고 파트너에 의해 수집 및 처리됩니다. 자세한 내용은 9항을 참조하세요.

3. 데이터 처리 및 저장

앱 데이터 처리:

  • 식단 데이터는 정부 공공 API(gbmo.go.kr)에서 가져옵니다
  • 즐겨찾기 설정(청사, 식당)은 기기 내 로컬에만 저장됩니다 (SharedPreferences)
  • 개인정보는 외부 서버에 수집하거나 저장하지 않습니다
  • 앱을 삭제하면 로컬에 저장된 설정도 함께 삭제됩니다

광고 데이터 처리:

  • 광고 ID 및 관련 데이터는 Google 광고 서버로 전송됩니다
  • 광고 상호작용 데이터는 Google과 파트너사에서 처리됩니다
  • 이 데이터는 Google의 보관 정책에 따라 저장됩니다 (9항 참조)

4. 데이터 공유 및 제3자 제공

앱 데이터 공유:

  • 정부청사 구내식당 앱은 개인정보를 제3자와 공유하지 않습니다

광고 데이터 공유:

Google Mobile Ads와 광고 파트너는 광고 목적으로 데이터를 공유할 수 있습니다:

  • 광고 ID(AD_ID)는 최대 141개 TCF 벤더 및 69개 광고 파트너와 공유될 수 있습니다
  • 광고 성과 지표는 캠페인 최적화를 위해 공유될 수 있습니다
  • 기기 정보는 광고 타겟팅을 위해 공유될 수 있습니다

앱 내 동의 대화상자를 통해 데이터 공유에 대한 동의를 관리할 수 있습니다 (9항 참조).

5. 앱 권한

본 앱은 최소한의 권한만 사용합니다:

  • INTERNET: 정부 API에서 식단 데이터를 가져오고 광고를 표시하기 위해 사용
  • AD_ID: 광고 캠페인 분석 목적으로만 사용

개인정보, 기기 저장소, 카메라, 마이크 또는 기타 개인 데이터에 접근하는 권한은 요청하지 않습니다.

6. 이용자 권리

앱에서 개인정보를 수집하지 않으므로:

  • 삭제 요청이 필요하지 않습니다
  • 정보 수정 또는 접근 요청이 적용되지 않습니다
  • 데이터 이동권이 적용되지 않습니다

GDPR 및 광고 관련 권리:

  • 맞춤형 광고에 동의하거나 거부할 권리
  • 개별 광고 파트너 설정을 관리할 권리
  • 언제든지 동의를 철회할 권리
  • 데이터 처리에 대한 정보에 접근할 권리

이러한 권리를 행사하려면 앱 내 동의 관리 대화상자를 이용하거나 직접 문의하시기 바랍니다.

7. 보안

본 앱은 다음과 같은 보안 조치를 구현합니다:

  • 모든 네트워크 통신은 보안 HTTPS 연결을 사용합니다
  • 식단 데이터는 신뢰할 수 있는 정부 공공 API에서만 가져옵니다
  • 로컬 데이터(즐겨찾기)는 기기의 보안 정책을 따릅니다
  • 개인정보는 외부로 전송하거나 저장하지 않습니다

8. 데이터 안전

데이터 수집 요약

  • 앱 자체: 데이터를 수집하지 않음
  • 광고 SDK: 광고 ID(AD_ID), 기기 정보, 광고 상호작용 데이터 수집

제3자 데이터 공유

  • 앱 자체: 데이터를 공유하지 않음
  • 광고 SDK: 광고 목적으로 최대 141개 TCF 벤더 및 69개 광고 파트너와 데이터 공유 가능

데이터 사용 목적

  • 광고 ID: 광고 맞춤화 및 캠페인 측정에 사용
  • 기기 정보: 광고 타겟팅 및 호환성에 사용
  • 광고 상호작용 데이터: 광고 성과 측정에 사용

사용자 제어

  • 동의 대화상자를 통해 맞춤형 광고를 거부할 수 있습니다
  • 개별 광고 파트너 설정을 관리할 수 있습니다
  • 기기 설정에서 광고 ID를 재설정할 수 있습니다

보안

  • 업계 표준 보안 모범 사례를 따릅니다
  • OWASP 가이드라인 준수
  • 보안 연결(HTTPS)을 통한 데이터 전송

9. 광고 및 Google Mobile Ads

중요: 본 앱은 Google Mobile Ads를 사용하여 광고를 표시합니다. 정부청사 구내식당 앱 자체는 개인정보를 수집하지 않지만, 광고 SDK는 아래 설명된 대로 데이터를 수집하고 처리합니다.

Google Mobile Ads에서 수집하는 데이터:

  • 광고 ID (AD_ID): 광고 목적의 고유한 재설정 가능 식별자
  • 기기 정보: 기기 모델, OS 버전, 화면 크기, 언어 설정
  • 광고 상호작용 데이터: 광고 노출, 클릭, 전환
  • IP 주소: 대략적인 지리적 위치 파악에 사용
  • 앱 패키지명 및 버전

데이터 수집 목적:

  • 사용자에게 관련성 있는 광고 제공
  • 광고 캠페인 성과 및 효과 측정
  • 사기 방지 및 광고 품질 보장
  • 법적 의무 준수

제3자 광고 파트너:

Google Mobile Ads는 광고 ID 및 관련 데이터를 다음과 공유할 수 있습니다:

  • 최대 141개 TCF (투명성 및 동의 프레임워크) 벤더
  • 최대 69개 Google 인증 광고 파트너

이 파트너들은 맞춤형 광고를 제공하고 광고 성과를 측정하기 위해 데이터를 사용합니다. 각 파트너는 자체 개인정보처리방침에 따라 운영됩니다.

귀하의 권리와 선택:

  • 동의 대화상자를 통해 맞춤형 광고에 동의하거나 거부
  • 개별 벤더 및 파트너 설정 관리
  • 앱 설정을 통해 언제든지 동의 철회
  • 기기 설정에서 광고 ID 재설정 (Android: 설정 → Google → 광고)
  • 시스템 전체에서 관심 기반 광고 거부

데이터 보관:

Google과 광고 파트너는 자체 보관 정책에 따라 광고 데이터를 보관하며, 일반적으로:

  • 광고 노출 데이터: 최대 24개월
  • 광고 ID 연결: 사용자가 ID를 재설정하거나 거부할 때까지

추가 자료:

  • Google 개인정보처리방침
  • Google 광고 개인정보
  • Google 광고 설정 관리

광고 파트너 및 TCF 벤더의 전체 목록은 앱 내 동의 관리 대화상자에서 확인할 수 있습니다.

10. 아동 개인정보 보호

앱의 데이터 수집:

정부청사 구내식당 앱은 13세 미만 아동을 포함하여 모든 연령의 사용자로부터 개인정보를 수집하지 않습니다.

광고와 아동:

Google Mobile Ads는 아동을 포함한 모든 사용자로부터 광고 ID(AD_ID)와 기기 정보를 수집할 수 있습니다. 단:

  • 아동에게 표시되는 광고는 연령에 적합하도록 필터링됩니다
  • 동의 설정을 통해 맞춤형 광고를 비활성화할 수 있습니다
  • 부모는 기기 설정에서 광고 ID를 재설정할 수 있습니다

본 앱은 COPPA (아동 온라인 개인정보 보호법) 및 Google Play의 가족 정책 요구 사항을 준수합니다.

12. 보안 검토 및 인증

본 앱은 높은 보안 표준을 유지합니다:

  • 정기적인 보안 모범 사례 구현

  • 일반적인 모바일 애플리케이션 취약점으로부터 보호
